# How you can test your Node.js applications with Jest

https://jestjs.io/docs/en/getting-started.html

```
mkdir tuts-jest && cd $_
```

### Modules installed
 - npm init -y
 - npm i jest --save-dev


### Setup ( set jest as ur test  in pck.json )
 - "scripts": { "test": "jest" }

### Running tests
 - yarn test
 - yarn
