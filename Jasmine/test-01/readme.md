# How you can test your Node.js applications with Jasmine

https://jasmine.github.io/pages/getting_started.html


### Modules installed
 - npm install --save-dev jasmine

### Init
 - npx jasmine init

### Setup ( set jasmine as ur test  in pck.json )
 - "scripts": { "test": "jasmine" }
 
### Running server 
 - npm start

### Running tests
 - npm test
